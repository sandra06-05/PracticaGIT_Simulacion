Realizar un mtodo que permita generar nmeros pseudoaleatorios pro medio de los cuadrados medios

Se buscar encontrar la eficiencia de la generacin de nmeros pseudo-aletorios a travs de los mtodos de cuadrados medios y congruencia lineal, para ello se debe seguir el siguiente proceso:

  1. A traves de la misma api generar una semilla diferente.

  2. Encontrar el numero de iteraciones hasta que se repita uno de sus datos.

  3. Generar 100 simulaciones con diferentes semillas.

  4. Generar un histograma con el resultado obtenidos por cada mtodo.

  5. Agregar sus conclusiones, opiniones y recomendaciones

Conclusiones¶

  • Como se observa en cada uno de los histogramas podemos ver que en el primer mtodo la semilla tendr mucho que ver con los resultados que se esperan, puesto que en el primer ejemplo se trabaja con una secuencia de semilla, pero en el segundo ejemplo se altera un poco la semilla por tanto los resultados como muestra la grfica son mucho mejores que los primeros.
  • Para con el segundo mtodo es un poco mas sencillo el proceso por parte de la semilla, pero se debe tener en cuenta que existen otras variables que intervienen en el proceso, mismas que con una nueva semilla deberan cambiar para un mejor resultado.

Opiniones¶

  • Cada mtodo tiene su forma de encontrar nmeros aleatorios, creo que se puede usar cada mtodo segn se necesita, ya que a mi parece el segundo mtodo al tener mas variables que interactuan es mejor para la generacin de estos nmeros, sin embargo el primer mtodo ser ptimo siempre y cuando la semilla sea la mejor.

Recomendaciones¶

  • Seleccionar una semilla adecuada para iniciar el proceso, para de esta manera evitar que los nmeros que el mtodo genere se repitan ciclicamente luego de mnimas iteraciones.
  • En caso de usar un mtodo para la generacin de una semilla, probarlo con un nmero considerable de iteraciones para de esta manera saber que tanto se repiten los nmeros a la hora de ser generados y hacer uso de estos.
  • Para elegir una semilla se puede usar la hora del ordenador, o tambien la temperatura del CPU entre otros elementos.
